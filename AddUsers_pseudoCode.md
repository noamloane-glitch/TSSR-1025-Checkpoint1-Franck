

Nom du script **addUsers.sh**

Shebang **#!/bin/bash**

1. Vérifier la présence des arguments
	1.1 Si ok on continue la création du  utilisateur
	1.2 Si ko message erreur et on sort du script
	
2. A chaque utilisateur on doit vérifier si le compte existe 
	2.1 Si ok on continue
	2.2 si ko affichage d'un message et on continue 

3. Création du compte utilsateur 
	3.1 On verifie si le compte a bien été créé
		3.1.1 Si ok affichage d'un message
		3.1.2 Si ko affichage d'un message erreur
4. Fin du script
